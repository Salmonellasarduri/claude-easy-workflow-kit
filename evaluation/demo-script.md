## 6:00 — 参謀の正体（2分）

> 「選んで、はい、で終わり」だと便利ツールで終わるんですけど、
> 本当にすごいのは /strategy_deep っていうモードです。
>
> これを使うと、Claude だけじゃなくて、
> 裏で Codex（GPT系）と Gemini も動いて、3方向から案を叩くんです。
>
> 実際にあった話をすると——
> 「AIの設定ファイルが古い記憶に固着してる」って問題を投げたんですね。
>
> Claude が「古いやつを消しましょう」って案を出した。
> そしたら Codex が「消すと人格の中核まで壊れますよ」って指摘した。
> さらに Gemini が「全部きれいにまとめると、つまらない優等生になりますよ」って言った。
>
> 1人では絶対気づかない穴です。
> 3ラウンド回したら、最初の案とは全然違う、もっといい設計が出てきた。
>
> ざっくり方向だけ示したら、参謀が勝手にレールを敷いてくれる。
> しかも穴まで見つけてくれる。

## 質疑で聞かれそうなこと（準備）

| 質問 | 回答 |
|------|------|
| superpowers との違いは？ | superpowers は開発者の創造性拡張、CEWK はあいまいな指示をレールに変える。競合じゃない |
| /strategy_deep は毎回使うの？ | 設計判断が必要なときだけ。簡単なタスクは /strategy の1ショットで十分 |
| Codex/Gemini は必須？ | オプション。なくても Claude 単体で動く。あると視点が増える |
| Windows でも動く？ | 動く。scaffold.ps1 がある |
| 無料？ | MIT ライセンス。Claude Code の利用料だけ |
