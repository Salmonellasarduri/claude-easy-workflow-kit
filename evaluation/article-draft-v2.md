# やりたいことはある。でも何から手をつければ？ → AIに /strategy って打ったら参謀が現れた

---

やりたいことリストだけは長い。

AIキャラ作りたい、配信をAIでやりたい、自分だけのツール作りたい。
でも「何から手をつけるか」で止まってる。

AIに聞いてみると、一瞬で何か出てくる。
でもそれを見て「……これでいいの？」ってなる。

案は出る。でも **比較できない。抜け漏れに気づけない。どれを採用すべきか決めきれない。**

---

## /strategy って打ったらこうなった

Claude Code で `/strategy カードゲームを作って` と打ってみた。

すると3つの案が出てきた。

> **A: シングルHTML — ブラウザで即プレイ**
> **B: Python CLI — ターミナルで対戦**
> **C: React+Vite — モダンWebアプリ**
> おすすめ: A

「A」と打ったら、自動で計画が出てきた。
「はい」と打ったら、実装が始まった。
テストが走って、レビューが通って、「保存しますか？」と聞かれた。

**自分が打ったのは「A」と「はい」と「保存」の3回だけ。**

仕様通りのカードゲームが、テスト済みで、記録付きで出てきた。

## 比べてみた

同じ「カードゲーム作って」を、3つの環境で試した。

| | 素の Claude Code | superpowers | CEWK |
|---|---|---|---|
| 完成まで | ~1分 | ~8分 | ~4分 |
| ユーザーの操作 | 1回（プロンプトだけ） | 3回 | 4回 |
| 仕様通りにできたか | ○ | ×（別ゲームになった） | ○ |
| テスト・レビュー | なし | なし | あり |
| 「なぜこの作り方？」が分かるか | × | × | ○ |

素の Claude Code は最速。でも全部お任せで、なぜこうなったか分からない。

superpowers（GitHub 130k スターの有名ツール）は5個のアイデアを出してくれた。楽しい。でも選んだら頼んだのと違うゲームができた。

CEWK は「選んで、はい、で完成」。しかも途中でテスト8項目、レビュー9項目が自動で走る。

**解いてる問題が違う。**

| やりたいこと | おすすめ |
|-------------|---------|
| とにかく速く | 素の Claude Code |
| 何を作るか一緒に考えたい | superpowers |
| **ざっくり言えば、あとはレール敷いてほしい** | **CEWK** |

## 参謀はもう1段深い

/strategy は「1回で案を出す」モード。

もう1つ、**/strategy_deep** というモードがある。
これを使うと、**見落としを別の視点から潰してくれる。** Claude だけじゃなくて、Codex と Gemini も裏で動いて、3方向から案を叩く。

実際にあった話:

AIの設定ファイルが古い記憶に固着している、という問題を投げた。

- Claude が「古いやつを消しましょう」と提案した
- Codex が **「消すと人格の中核まで壊れますよ」** と指摘した
- Gemini が **「全部きれいにまとめると、つまらない優等生になりますよ」** と指摘した

1人では絶対気づかない穴。3ラウンド回したら、最初とは全然違う設計が出てきた。

ざっくり方向を示すだけで、参謀がレールを敷いてくれる。しかも穴まで見つけてくれる。

## 始め方

Claude Code で:

```
claude-easy-workflow-kit を入れて
```

入ったら:

```
/strategy やりたいこと1行
```

以上。あとは聞かれたことに答えるだけ。

**先送りしてる作業、1個あると思う。**
**それを参謀に渡してみてほしい。**
